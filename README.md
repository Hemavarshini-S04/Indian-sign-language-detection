# Indian Sign Language Detection using YOLO

## 📌 Project Overview
This project detects Indian Sign Language gestures using a YOLO-based deep learning model.

## 🚀 Features
- Real-time hand gesture detection
- Uses trained YOLO model
- Web interface support

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLO (Ultralytics)
- Flask

## 📂 Dataset
Custom dataset trained using Roboflow

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
